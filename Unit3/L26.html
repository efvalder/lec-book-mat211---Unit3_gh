9.1 Definition of a Periodic function

NoteDefinition

A function \(f(t)\) defined for \(t\geq 0\) is periodic with period \(T>0\) if \[f(t+T)=f(t),\qquad \forall t \geq 0\]

9.2 Theorem: Laplace of a periodic function

NoteTheorem

Theorem 9.1 Let \(f(t)\) be piecewise continuous, periodic T, and of exponential order for \(t\geq 0\), then

\[\boxed{\mathscr{L}\{f(t)\}=\frac{1}{1-e^{-sT}}\int_0^T e^{-st}f(t)\,dt} \tag{9.1}\]

CautionProof:

\[\mathscr{L}\{f(t)\}=\int_0^\infty e^{-st} f(t)\,dt\]

\[=\int_0^T e^{-st} f(t)\,dt+\int_T^\infty e^{-st} f(t)\,dt\]

Change of variables: \[v=t-T\rightarrow v(T)=0,\quad dv=dt\]

\[=\int_0^T e^{-st} f(t)\,dt+\int_{v=0}^\infty e^{-s(v+T)} f(v+T)\,dv\] and because the period is \(T\), then \(f(v+T)=f(v)\)

\[=\int_0^T e^{-st} f(t)\,dt+e^{-sT}\int_0^\infty e^{-sv} f(v)\,dv\] \[\mathscr{L}\{f(t)\}=\int_0^T e^{-st} f(t)\,dt+e^{-sT}\mathscr{L}\{f(t)\}\] \[\Rightarrow \mathscr{L}\{f(t)\}\left(1-e^{-sT}\right)=\int_0^T e^{-st} f(t)\,dt\]

\[\Rightarrow \mathscr{L}\{f(t)\}=\frac{1}{\left(1-e^{-sT}\right)}\int_0^T e^{-st} f(t)\,dt\]

\[\hspace{2in}\square\]

9.2.1 Example 1

TipExample

Let \(f(t+2)=f(t)\)

and the periodic function defined by:

\[ f(t) = \begin{cases} +1 & 0\leq t < 1 \\ -1 & 1\leq t < 2 \end{cases} \]

Calculate \(\mathscr{L}\{f(t)\}\)

CautionSolution:

\[\mathscr{L}\{f(t)\}=\frac{1}{\left(1-e^{-sT}\right)}\int_0^T e^{-st} f(t)\,dt\]

\[=\frac{1}{\left(1-e^{-2s}\right)}\left(\int_0^1 e^{-st} (1)\,dt+\int_1^2 e^{-st} (-1)\,dt\right)\] \[=\frac{1}{\left(1-e^{-2s}\right)}\left(\frac{e^{-st}}{-s}\Big|_0^1-\frac{e^{-st}}{-s}\Big|_1^2\right)\] \[=\frac{1}{\left(1-e^{-2s}\right)}\left(\frac{-1}{s}\right)\left(e^{-s}-e^{0}-(e^{-2s}-e^{-s})\right)\]

\[=\frac{1}{\left(1-e^{-2s}\right)}\left(\frac{-1}{s}\right)\left(2e^{-s}-1-e^{-2s}\right)\]

\[=\frac{1}{\left(1-e^{-2s}\right)}\left(\frac{1}{s}\right)\left(1+e^{-2s}-2e^{-s}\right)\]

9.2.2 Example 2

TipExample

\[y''-3y'+2y= f(t),\qquad y(0)=y'(0)=0\] Where:

\[ f(t) = \begin{cases} +1 & 0\leq t < 1 \\ -1 & 1\leq t < 2 \end{cases} \] \[f(t+2)=f(t)\]

CautionSolution:

Same periodic function of last example, therefore we know the Laplace Transformation of the RHS, for the LHS, we use properties:

\[s^2Y-sy(0)-y'(0)-3(sY-y(0))+2Y= \frac{1}{\left(1-e^{-2s}\right)}\left(\frac{1}{s}\right)\left(1+e^{-2s}-2e^{-s}\right)\]

\[s^2Y-3sY+2Y= \frac{1}{\left(1-e^{-2s}\right)}\left(\frac{1}{s}\right)\left(1+e^{-2s}-2e^{-s}\right)\]

\[Y(s^2-3s+2)= \frac{1}{\left(1-e^{-2s}\right)}\left(\frac{1}{s}\right)\left(1+e^{-2s}-2e^{-s}\right)\]

\[Y(s-1)(s-2)= \frac{1}{\left(1-e^{-2s}\right)}\left(\frac{1}{s}\right)\left(1+e^{-2s}-2e^{-s}\right)\]

\[Y= \frac{\left(1-2e^{-s}+e^{-2s}\right)}{\left(1-e^{-2s}\right)}\left(\frac{1}{s}\right)\frac{1}{(s-1)(s-2)}\] \[Y= \frac{(1-e^{-s})(1-e^{-s})}{(1-e^{-s})(1+e^{-s})}\left(\frac{1}{s}\right)\frac{1}{(s-1)(s-2)}\] \[Y= \frac{(1-e^{-s})}{(1+e^{-s})}\left(\frac{1}{s(s-1)(s-2)}\right)\]

Partial fractions for the right side factor!

\[\left(\frac{1}{s(s-1)(s-2)}\right)=\frac{A}{s}+\frac{B}{(s-1)}+\frac{C}{(s-2)}\]

\[1=A(s-1)(s-2)+Bs(s-2)+Cs(s-1)\] \(\boxed{s=0}\Rightarrow\)

\[1=A(-1)(-2)\rightarrow \boxed{A=\frac{1}{2}}\] \(\boxed{s=1}\Rightarrow\)

\[1=B(1)(-1)\rightarrow \boxed{B=-1}\] \(\boxed{s=2}\Rightarrow\)

\[1=C(2)(1)\rightarrow \boxed{C=\frac{1}{2}}\]

Therefore:

\[Y=\frac{(1-e^{-s})}{(1+e^{-s})}\left(\frac{1}{2s}-\frac{1}{s-1}+\frac{1}{2(s-2)}\right)\] \[Y(1+e^{-s})=(1-e^{-s})\left(\frac{1}{2s}-\frac{1}{s-1}+\frac{1}{2(s-2)}\right)\] \[Y+Ye^{-s}=\left(\frac{1}{2s}-\frac{1}{s-1}+\frac{1}{2(s-2)}\right)-e^{-s}\left(\frac{1}{2s}-\frac{1}{s-1}+\frac{1}{2(s-2)}\right)\]

We know that:

\[\mathscr{L}\{u(t-a)f(t-a)\}=e^{-as}F(s)\]

\[\Rightarrow\boxed{\mathscr{L}^{-1}\{e^{-as}F(s)\}=u(t-a)f(t-a)}\] Then applying ILT to:

\[Y+e^{-s}Y=\left(\frac{1}{2s}-\frac{1}{s-1}+\frac{1}{2(s-2)}\right)-e^{-s}\left(\frac{1}{2s}-\frac{1}{s-1}+\frac{1}{2(s-2)}\right)\]

we obtain:

\[\boxed{y(t)+u(t-1)y(t-1)=\frac{1}{2} - e^{t}+\frac{1}{2}e^{2t}-\left(\frac{1}{2}u(t-1)-u(t-1)e^{t-1}+\frac{1}{2}u(t-1)e^{2(t-1)}\right)}\]
